# 🌐 Personal Portfolio Website

A modern, responsive, and visually engaging personal portfolio website built to showcase projects, skills, certifications, achievements, and professional experience. This portfolio is designed with clean UI principles, smooth navigation, and structured sections for an enhanced user experience.

---

## 📂 Project Structure

```bash
portfolio/
│
├── assets/                 # Images, icons, styles, scripts, etc.
│
├── home.html               # Landing / homepage
├── about.html              # About me section
├── skills.html             # Technical & professional skills
├── project.html            # Projects showcase
├── certifications.html     # Certifications & achievements
├── prizes.html             # Awards & recognitions
├── resume.html             # Resume display/download
├── contact.html            # Contact form & details
├── login.html              # Authentication page
├── admin.html              # Admin dashboard/page
│
└── README.md               # Project documentation
```
